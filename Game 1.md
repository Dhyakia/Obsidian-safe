
# PowerGrid Iddler (titre non définitif)

## Thème

Un clicker / iddler sur le thème de l'énergie.
Le joueur commence en devant clicker pour générer une étincelle jusqu'à générer l'intégralité de l'energie présente sur la Terre. Le gag est de feinté l'existence d'un menu "renaissance" alors qu'il n'existe qu'une seul Terre :d

## Concept

Le joueur doit faire les mécaniques permet au joueur de générer de l'énergie permettant d'acheter des upgrades. Donc je doit designer disons 6-10 mécaniques différentes pour générer de l'anergie jusqu'à pouvoir l'automatiser.

Bois < Force Animale < Hydraulique < Eolienne < Charbon < Pétrole < Gaz naturel < Nucléaire < Solaire < Geothermie


Comme dans un iddler classique, les upgrades sont : 
- Plus de générateur
- Automatisation
- Multiplicateur de l'objet ou global
- ... et autre :)

Le joueur reçois des hints sur les breakpoints : 
- "X énergie correspond a un micro-onde" 
- "Y correspond a l'énergie consommé par jour par x pays"
- ... et autres !

Je vise un temps de complétion de 6h.

## Objectifs

Produire un prototype rapidement et **suivre les conseils d'Hakita** : 
- Rapidement créer un prototype complet
- Se concentrer sur la boucle de jeux principale

Une fois le prototype terminé, il est très important de **mixer les arts** ; ne pas attendre d'avoir rédigé toutes la logique pour lancer les animations et le son / bruitage. Terminer un objet dans son intégralité n'est pas le mieux en terme d'efficacité mais cela permet de se projeter beaucoup plus facilement et donc et ne pas avoir à refaire des éléments mal cadré.
